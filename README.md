# Менеджер задач

В проекте реализовано:
- добавление задачи;
- ввод названия и дедлайна;
- статусы **To Do**, **In Progress**, **Done**;
- подсветка просроченных задач;
- фильтр по статусу;
- отображение текущей даты.

## Как запустить проект

1. Установите **Node.js**.
2. Откройте терминал в папке проекта.
3. Выполните команды:

```bash
npm install
npm run dev
```

4. После запуска откройте ссылку, которую покажет терминал.
5. Для итоговой сборки можно использовать:

```bash
npm run build
```

## Как пользоваться приложением

1. Введите название задачи.
2. Выберите дедлайн.
3. Нажмите **«Добавить»**.
4. Новая задача появится в списке.
5. Для каждой задачи можно менять статус через выпадающий список.
6. Если дедлайн уже прошёл, а задача ещё не имеет статус **Done**, она будет подсвечена как просроченная.
7. В верхней части страницы можно отфильтровать задачи по статусу.
